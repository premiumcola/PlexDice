## P1 — Bugs (broken/incorrect behavior)

- **`frontend/src/lib/plexLink.js:35-49` · `openInPlexApp` web-fallback false-positive · FIXED.**
  The 1.5 s fallback only cleared on `visibilitychange`. On iOS the native app
  takeover sometimes fires `pagehide`/`blur` *without* a `visibilitychange`, so the
  timer still ran and opened Plex Web in Safari on top of the launched app.
  Fix: clear the timer on `visibilitychange` (hidden) **and** `pagehide` **and**
  `blur`, with a guard flag + listener cleanup so it fires at most once.

## P2 — Code smells (dead code, duplication, types, names)

- **`backend/routes/movie_info.py:30-35` + `backend/quiz/history.py:40-44` ·
  duplicated atomic-write logic · FIXED.** Both reimplement the tmp-file +
  `os.replace` pattern that already lives in `atomic_io.atomic_write_json`. Routed
  both through `atomic_write_json` (single source of truth; also gains fsync +
  keep-old-file-on-failure — see P3).
- **Flask view functions omit return-type hints (e.g. `routes/*.py` `def info():`)
  · Deferred.** Conventional for Flask handlers returning `Response`/tuple; not
  changed (no behavior impact, out of scope for a no-feature pass).

## P3 — Hardening (error handling, edge cases, log noise)

- **`backend/routes/movie_info.py` / `backend/quiz/history.py` cache writes lacked
  `fsync` and crash-safety · FIXED** by routing through `atomic_write_json`, which
  fsyncs the tmp file and leaves the existing file intact on any `OSError` (a failed
  write can no longer truncate the cache).
- **`frontend/src/lib/plexLink.js:27-30` · native deep-link routing not verifiable
  headless · CAVEAT (no code change).** The URL shape is correct —
  `plex://preplay/?metadataKey=%2Flibrary%2Fmetadata%2F<ratingKey>&server=<machineId>`,
  `metadataKey` encoded exactly once, `server` guarded non-empty (the library
  server's `machineIdentifier`, not account-level). Whether it lands on the movie's
  preplay screen vs. the app home depends on the installed Plex iOS app and can only
  be confirmed on a physical device.
